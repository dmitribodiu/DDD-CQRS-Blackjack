To run the Acceptance tests that use WatiN using IE8 you will have to do one of the following:

- Run Visual Studio in administrator mode or run the RunUATTests batch file in administrator mode.

- In Internet Explorer, open Tools -> Internet Options -> Security -> Local intranet and check the box "Enable Protected Mode."
 more info on this here http://spin.atomicobject.com/2010/12/15/watin-and-com-errors-enable-ie-protected-mode-for-local-intranet-zone
